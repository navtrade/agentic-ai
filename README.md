# Agentic-ai

## 项目简介
本项目围绕吴恩达老师（Andrew Ng）在 DeepLearning.AI 推出的 Agentic AI 系列课程，致力于为中文学习者提供高质量的课程内容翻译、系统化的知识梳理、关键概念解析以及配套示例代码的详细解读。项目不仅帮助学习者跨越语言障碍，更通过结构化整理与实践引导，深入理解“智能体”（Agent）在现代 AI 系统中的核心作用，掌握构建自主、协作、推理型 AI 应用的关键方法。

## 项目受众
- 对 Agentic AI 感兴趣
- 正在系统学习LLM高级应用
- 希望掌握 LLM 与本地工具集成
- 计划基于开源模型构建领域专用智能体
- 关注 AI 自动化流程 与 任务协作系统

## 项目亮点
- 精准的中英对照翻译：忠实还原原课程技术细节，兼顾术语准确性与中文表达流畅性
- 结构化知识图谱：将课程内容拆解为“概念 → 原理 → 架构 → 代码”四层知识体系，便于系统掌握
- 配套代码深度解读：不仅提供示例代码，更详解其设计思路、依赖库使用及可扩展性建议
- 结合本地工具实践场景：示例融入文件读写、API 调用、数据处理等实用功能
- 持续更新与社区共建：随课程迭代同步更新，欢迎贡献翻译、笔记或教程，共建中文 Agentic AI 学习生态
⭐ 欢迎 Star、Fork 与 PR，一起打造最好的 Agentic AI 学习资源！
